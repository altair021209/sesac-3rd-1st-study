*유클리드 호제법 : 두 수의 최대 공약수를 구하는 알고리즘

*******************************************************************************************************************************
두 정수 a, b의 최대공약수를 G(a, b)라고 하자.

정수 a, b, q, r (b ≠ 0)에 대하여 a = bq + r,이면 G(a, b) = G(b, r)가 성립한다.

〈증명〉
G(a, b) = g라고 하자. 최대공약수의 성질에 의해 a=a′g, b=b′g 이고 G(a′, b′) = 1이다.
a = bq + r로부터 r = a - bq = g(a′ - b′q) 이고, g는 r의 약수이다.
G(b, r) = g임을 보이기 위해서는 G(b′, a′ - b′q) = 1임을 보이면 된다.

귀류법을 적용하여 결론을 부정해보자.
어떤 정수 d가 존재하여 G(b′, a′-b′q) =d(≠ 1)라고 하면, b′ = dm,  a′-b′q = dn 라고 할 수 있고, 
a′ = b′q + dn = (dm)q + dn = d(mq + n) 이므로  
a'=d(mq+n)과 b'=dm은 공약수 d를 가져서 G(a′, b′) = 1라는 가정에 모순이다. 

따라서 G(b′, a′ - b′q) = 1이고 그래서 G(b, r) = g가 성립한다.
즉, G(a,b) = G(b,r) 이다.
*******************************************************************************************************************************

*코드 구형
```python
def gcd(a,b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b) #재귀함수

t = int(input())

for i in range(t):
    a, b = map(int, input().split())
    result = a * b // gcd(a, b)  #최소공배수 = 두 수의 곱 / 최대공약수
    print(int(result))
```
