# 문제 042 --- 소수 & 팰린드롬
## 문제 설명

정수 N이 주어지면 N 이상인 수들 중에서 소수이면서 동시에 팰린드롬인 가장 작은 값을 찾는 문제다.
소수는 에라토스테네스의 체를 이용해 미리 전체 범위를 구해두고,
팰린드롬은 숫자를 문자열로 만들어 양쪽을 비교하면서 확인한다.

핵심은 모든 수를 직접 검사하는 게 아니라
소수를 먼저 빠르게 걸러내고
그 다음 팰린드롬 여부를 확인해 효율적으로 답을 찾는 방식이다.

## 풀이 과정

먼저 1부터 1,000,000까지 소수 여부를 저장하는 리스트를 만든다.
그리고 N부터 하나씩 증가시키면서
현재 숫자가 소수인지 확인하고
소수라면 팰린드롬인지 확인한다.
두 조건을 모두 만족하면 바로 출력하고 끝낸다.

## 코드

``` python
import math
N = int(input())
A = [0] * (1000001)        # 소수 여부 저장 리스트

for i in range(2, len(A)):
    if A[i] == 0:          # 아직 걸러지지 않은 수 = 소수 후보
        for j in range(i + i, len(A), i):
            A[j] = 0       # 배수는 소수가 아님

def isPalindrome(target):
    temp = list(str(target))  # 문자열로 변환
    s = 0                     # 왼쪽
    e = len(temp) - 1         # 오른쪽
    while s <= e:
        if temp[s] != temp[e]:
            return False      # 하나라도 다르면 팰린드롬 X
        s += 1
        e -= 1
    return True               # 끝까지 같으면 팰린드롬

i = N

while True:
    if A[i] != 0:             # 소수인지 확인
        result = A[i]
        if isPalindrome(result):  # 팰린드롬인지 확인
            print(result)         # 조건 만족 → 출력
            break
    i += 1                      # 다음 숫자 확인
```
