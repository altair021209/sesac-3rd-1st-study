# 이항 계수 1

- 자연수 N과 정수 K가 주어졌을 때 이항 계수 $_nC_k$를 구하는 문제입니다.
- 이항 계수는 곧 **조합(Combination)**을 의미하며, N개의 수에서 K개를 순서 없이 뽑는 경우의 수입니다.
- N이 최대 10으로 매우 작기 때문에, 복잡한 점화식 없이 조합 공식(팩토리얼)을 그대로 구현해도 시간 내에 충분히 해결 가능합니다.

# 과정

- N과 K를 입력받음
- 조합 공식 $_nC_k = \frac{n!}{(n-k)!k!}$ 을 그대로 코드로 구현
- 팩토리얼 계산을 위해 반복문 사용

# 코드

```python
import sys
input = sys.stdin.readline

# 1. 입력 받기
N, K = map(int, input().split())

# 2. 팩토리얼 함수 정의
# (재귀함수나 math 라이브러리를 써도 되지만, 과정을 보여주기 위해 반복문으로 구현)
def factorial(n):
    result = 1
    for i in range(1, n + 1):
        result *= i
    return result

# 3. 조합 공식 적용: n! / ((n-k)! * k!)
numerator = factorial(N)            # 분자: n!
denominator = factorial(N - K) * factorial(K) # 분모: (n-k)! * k!

print(numerator // denominator)
