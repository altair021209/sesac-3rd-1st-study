# 부녀회장이 될테야

- 아파트의 k층 n호에 살기 위해서는, **(k-1)층의 1호부터 n호까지 사람들의 수의 합**만큼 데려와 살아야 합니다.
- 이 규칙은 정확히 **조합의 점화식(파스칼의 삼각형)** 원리와 같습니다.
- 빈 방(0층)부터 시작해서 사람 수를 채워나가는 바텀업(Bottom-Up) 방식의 DP 문제로 해석할 수 있습니다.

# 과정

- 테스트 케이스(T)만큼 반복
- 층수(k)와 호수(n) 입력
- 0층의 입주민 수 초기화 (1호=1명, 2호=2명, 3호=3명 ...)
- k층까지 올라가면서 아래층의 사람 수를 합산하여 현재 층을 갱신
  - **공식:** `현재 층 n호 사람 수` = `현재 층 (n-1)호 사람 수` + `아래 층 n호 사람 수` 
  - (혹은 문제 정의대로 아래층 1~n호의 합을 구해도 됨)

# 코드

```python
import sys
input = sys.stdin.readline

T = int(input())

for _ in range(T):
    # 1. k(층), n(호) 입력
    k = int(input())
    n = int(input())
    
    # 2. 0층 리스트 초기화 (0층의 i호에는 i명이 산다)
    people = [i for i in range(1, n + 1)]
    
    # 3. k층까지 반복
    for _ in range(k):
        # 각 층의 1호~n호까지 사람 수 갱신
        for i in range(1, n):
            # 현재 층의 i호 = (현재 층의 i-1호) + (아래 층의 i호 : 이미 people[i]에 저장된 값)
            # 사실상 people[i] = people[i] + people[i-1] 와 같음 (누적합)
            people[i] += people[i-1]
            
    # k층 n호의 사람 수 출력 (인덱스는 n-1)
    print(people[n-1])
